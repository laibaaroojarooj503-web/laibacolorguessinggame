# Color Guessing Game

## Description
A colorful and interactive **Color Guessing Game** using **HTML, CSS, and JavaScript**.  
Users need to guess the correct color from multiple choices. The background **dynamically changes** when the correct color is selected, making it visually appealing and interactive. This project demonstrates **advanced DOM manipulation, animations, and user interaction**.

## Features
- Six color options for guessing.
- Real-time feedback: correct / try again.
- Dynamic background change for visual effect.
- Interactive and colorful UI.

## Technologies Used
- HTML
- CSS (animations & gradients)
- JavaScript

## How to Play
1. Open `index.html` in a browser.
2. Look at the **color code** displayed.
3. Click on the button matching the color.
4. If correct, background and buttons turn into the correct color.
5. Click **New Colors** to reset the game.
